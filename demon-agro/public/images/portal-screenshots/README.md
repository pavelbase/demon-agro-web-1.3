# Portal Screenshots

Tato složka obsahuje screenshoty z portálu pro zobrazení na landing page.

## Požadované obrázky:

1. **health-card.png** - Screenshot zdravotní karty půdy
   - Ukazuje parametry pH, fosfor, draslík, hořčík, síru, vápník, poměr K:Mg
   
2. **parcels-list.png** - Screenshot seznamu pozemků
   - Tabulka s pozemky, jejich parametry a stavy
   
3. **liming-plan.png** - Screenshot časového plánu vápnění
   - Dlouhodobý plán s produkty a aplikacemi

## Formát:
- PNG nebo JPG
- Doporučená šířka: 1200-1600px
- Optimalizované pro web

## Nahrání:
Umístěte screenshoty ze své aplikace do této složky s výše uvedenými názvy.
